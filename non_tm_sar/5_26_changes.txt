_format_investigations_metadata_prompt.

The function signature:
def _filter_relevant_investigation_paragraphs(details_text: str) -> str:

Logic:
- Split details_text by double newline (paragraph boundary)
- Define a relevance keyword list: "suspicious", "unusual", "alert", 
  "negative news", "high-risk", "wire", "cash", "check", "ACH", "EFT", 
  "P2P", "Zelle", "Venmo", "CashApp", "bitcoin", "crypto", "ethereum", 
  "jurisdiction", any country name from the HIGH-RISK COUNTRIES LIST
- Keep a paragraph if it contains at least one keyword (case-insensitive match)
- Join and return the filtered paragraphs as a single string
- If no paragraphs match, return the first two paragraphs as fallback
